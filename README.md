# Amba Glass Company

A responsive, modern, and interactive website for **Amba Glass Company**, a premium glass installation and products provider.

## 🚀 Recent Updates

The website has been upgraded with a **Glassmorphism-inspired UI** and **Smooth Scroll Animations** to enhance the user experience and create a more premium feel. 

### Key Features Added:
- **Smooth Scrolling (`html { scroll-behavior: smooth; }`)**: Navigating via the header links now smoothly glides to the targeted sections.
- **Intersection Observer Animations (`script.js`)**: 
  - Elements now smoothly animate into view as you scroll down the page.
  - Custom animation classes include `.fade-up`, `.fade-in`, `.slide-left`, `.slide-right`, and `.scale-in`.
- **Dynamic Backgrounds**: Floating blurred orbs in the background create a deep, glass-like aesthetic.
- **Modern CSS Styling**: Upgraded the color palette, typography (Google Poppins), and hover effects across all buttons, cards, and images.
- **Glassmorphism Cards**: Product and Team cards feature a frosted-glass effect with subtle borders and shadows.

## 📁 File Structure

- **`index.html`**: The main structure of the website, including Header, Hero, Products, Team, Projects, Map, and Contact sections.
- **`style.css`**: Contains all the modern styling, responsive media queries, and the newly added scroll animation utility classes.
- **`script.js`**: A lightweight JavaScript file utilizing the `IntersectionObserver` API to trigger animations when elements scroll into the viewport.

## 💻 Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations, Backdrop-filter)
- Vanilla JavaScript (IntersectionObserver API)

## 🔧 Setup & Usage
Simply open `index.html` in any modern web browser to view the live website with all animations and styling. No build tools or dependencies are required.
